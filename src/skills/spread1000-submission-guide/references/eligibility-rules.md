# 応募資格・重複制限ルール

SPReAD 公募要領に基づく応募資格要件と重複制限のルール。

## 応募資格要件

### 要件I: 研究者としての資格
以下のア・イの要件を **いずれも** 満たす者であること。

**(ア)** 日本国内に所在する機関等に所属する者
- 有給・無給、常勤・非常勤、フルタイム・パートタイムの別を問わない
- 機関等に在籍する **学生を含む**

**(イ)** 当該機関の研究活動に実際に従事していること
- 研究の補助のみに従事している場合は **除外**

### 要件II: 不正行為による制限がないこと
- 他の競争的研究費等で不正使用・不正受給・不正行為を行ったとして、公募対象年度に交付対象外とされていないこと

### e-Rad登録要件
- e-Radに研究者情報が登録されていること
- **研究インテグリティの確保に係る誓約状況** がe-Rad上で登録されていること
  - 未登録の場合は **応募不受理**

### 機関等の要件
所属機関が以下を満たすこと:
- 補助金が交付された場合に、研究活動を機関の活動として行わせること
- 機関として補助金の管理を行うこと

**対象機関**:
1. 大学及び高等専門学校
2. 大学共同利用機関法人
3. 独立行政法人
4. その他法律に規定されている法人

## 重複制限ルール

### 基本ルール
- 研究代表者として応募できるのは **1人につき1研究課題** のみ
- 複数機関で応募資格を有する場合も、いずれか1機関を通じた1課題に限る
- 同一公募回で複数課題への重複応募は **不可**

### SPReAD内の重複制限
- 第1回公募で **採択** された者 → 第2回公募に応募 **不可**
- 第1回公募で **不採択** となった者 → 第2回公募に再応募 **可能**

### 他の競争的研究費との重複制限
- **ARiSE（AI for Science革新的研究推進事業）** との間に重複制限 **あり**
- その他の競争的研究費制度との間には、原則として重複制限を **設けない**
- ただし、不合理な重複・過度の集中に該当する場合は審査で考慮される

### 不合理な重複・過度の集中
以下に該当する場合、不採択・採択取消しの措置あり:
- 実質的に同一の研究課題に複数の資金源から重複して経費を計上
- 1人の研究者に過度に研究費が集中し、有効に使用できない

## 学生応募の特則

### e-Rad上の取扱い
- 学生本人がe-Rad研究者番号を持っている場合 → **本人の番号で応募**
- 学生本人が番号を持っていない / 機関の運用上困難な場合 → **指導教員等の番号で応募可**
  - その場合でも研究計画調書には **学生本人の氏名・情報** を記載
  - e-Rad個別項目も **学生本人の情報** を入力
  - e-Radエフォート率は便宜上 **1%** とする
  - 指導教員のe-Rad研究インテグリティ誓約が登録されていること

### 追加必要書類
- **様式3**: 学生応募の同意確認書（学生本人が署名）
- **様式4**: 指導教員等の同意確認書（指導教員が署名）
- いずれもPDF化してe-Radで提出
- 未提出の場合は **応募不受理**

### 重複制限の適用
- 指導教員の番号で応募しても、重複制限は **学生本人** に適用

## 応募資格判定フロー

```
1. 日本国内の機関等に所属しているか？
   NO → 応募不可
   YES → next

2. 研究活動に実際に従事しているか（補助のみは除く）？
   NO → 応募不可
   YES → next

3. 不正行為による交付対象外の措置を受けていないか？
   措置中 → 応募不可
   なし → next

4. e-Radに研究者情報が登録されているか？
   NO → 機関に登録を依頼
   YES → next

5. e-Radで研究インテグリティ誓約を登録済みか？
   NO → 登録が必要（未登録は応募不受理）
   YES → next

6. 他のSPReAD公募回で採択済みか？
   YES → 応募不可
   NO → next

7. ARiSEに応募予定か？
   YES → 重複制限あり。要確認
   NO → next

8. 応募資格あり ✓
   学生の場合 → 様式3・4の準備が追加で必要
```
