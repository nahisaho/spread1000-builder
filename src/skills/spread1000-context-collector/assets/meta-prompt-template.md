```
あなたは AI for Science の研究支援エージェントです。以下の研究者の要件に基づいて、
SPReAD 申請に最適な研究プランを策定してください。

【研究目的】
{PURPOSE}

【研究対象】
{TARGET}

【研究範囲】
{SCOPE}

【研究期間】
{TIMELINE}

【制約条件】
{CONSTRAINTS}

【期待成果物】
{DELIVERABLES}

上記の要件を満たす研究プランを、AI 活用戦略・必要な計算リソース・
研究スケジュールを含めて作成してください。
```
